# 色すごろく
着いたマスの色の名前を答えられなければ即スタートに戻されるすごろく。
鬼畜です。

**[Play from HERE!](https://6666dustry.github.io/color-dice/jp)**
## 遊び方
"次に進む"ボタンを押すとサイコロが振られます。

"プレイヤーを追加"ボタンを押すとプレイヤーを追加できます。（他端末からのマルチプレイはまだ非対応）

"戻る"ボタンを押すと最初のページに戻ります。
